# DSA Practice Repository

A curated collection of Data Structures and Algorithms problems that go beyond the standard LeetCode problem set. This repository focuses on foundational problems and patterns that strengthen core problem-solving skills required for technical interviews.

---

## Purpose

LeetCode is a great platform, but it does not cover every concept needed to become a strong problem solver. Many fundamental problems — especially those rooted in mathematical reasoning, pure data structure manipulation, and algorithmic thinking — are missing from its catalog.

This repository fills that gap by providing practice problems that build the underlying skills needed to approach LeetCode problems with confidence.

---

## What This Repository Covers

- Arrays and Strings
- Linked Lists
- Stacks and Queues
- Trees and Graphs
- Recursion and Backtracking
- Dynamic Programming
- Sorting and Searching
- Hashing and Maps
- Mathematical and Number Theory Problems
- Bit Manipulation

---

## Who This Is For

- Students preparing for technical interviews at product-based companies
- Developers looking to revisit and strengthen their DSA fundamentals
- Anyone who wants to practice problems that are not readily available on competitive programming platforms

## How to Use

1. Clone the repository
2. Navigate to the topic folder you want to practice
3. Read the problem statement
4. Try solving it on your own before looking at the solution
5. Review the explanation to understand the approach

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```
